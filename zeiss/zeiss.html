Problem & Solution

렌즈 이미지가 너무 가까이 있는거 아니에요?

이미지 크기도 큰데 img:hover{ transform: scale(1.1); }를 적용하니hover시 화면에 너무 가득차서 좁아 보이는 문제점 발생 img-size 조절, img를 감싸고 있는 div에 width: 400px;, height: 500px;, overflow: hidden; 추가img에 width: 100%;, height: 100%;, object-fit: cover; 적용 후 해결

hover는 자연스럽게

global버튼 hover시 opacity값과 hover시 스타일 값 때문에 부자연스러움 global버튼,hover의 스타일 값을 width: 140px로 동일하게 맞추고transition: opacity 0.1s;을 주어 자연스럽게 변경되도록 함

항상 개발자도구를 활용하자

swiper의 autoplay-progress가 보이지 않음 개발자 도구로 확인해보니 위치값이 없음 position: absolute; top: 10%; right: 3%;, z-index: 999;을 적용하여 나타냄
